About Checkbox Cell Hider

Checkbox Cell Hider is an add-on for the Google Sheets™ web-based spreadsheet program. The add-on lets you hide cells with checkboxes (actually, it just makes the font color white). This tool is useful for creating "spreadsheet flashcards" with Google Sheets™.

Purpose

The intended purpose of this app (it's more like a simple script, really) is to help streamline this study technique by reducing the number of clicks needed to hide a cell.

How to Use

To start using Checkbox Cell Hider, simply install the add-on from the Google Workspace™ Marketplace and select Checkbox Cell Hider from the add-on menu in Google Sheets™, which will run the script.

Every time you run the script, it adds 100 checkboxes. Clicking a checkbox blanks the cell to its right. Running the script again will generate 100 more checkboxes below.

Attention

This add-on works best on blank documents.
