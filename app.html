About Checkbox Cell Hider

Checkbox Cell Hider is a Google Sheets add-on that lets you hide cells with checkboxes (actually, it just makes the font color white). This tool is useful for creating "spreadsheet flashcards" with Google Sheets.

Purpose

The intended purpose of this app (it's more like a simple script, really) is to help streamline this study technique by reducing the number of clicks needed to hide a cell.

How to Use

To start using Checkbox Cell Hider, simply install the add-on from the Google Workspace Marketplace and follow the instructions provided in the add-on menu in your Google Sheets.
