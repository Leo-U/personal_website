About Checkbox Cell Hider

Checkbox Cell Hider is an add-on for the Google Sheets™ web-based spreadsheet program. The add-on lets you hide cell content by clicking checkboxes (actually, it just makes the font color white). This tool is useful for creating "spreadsheet flashcards" with Google Sheets™.

Purpose

The intended purpose of this app (it's more like a simple script, really) is to help streamline this study technique by reducing the number of clicks needed to hide a cell.

How to Use

To start using Checkbox Cell Hider, the first step is to simply install the add-on from the Google Workspace™ Marketplace. The second step is to run the add-on. To run the add-on, from the add-on menu in Google Sheets™, select Checkbox Cell Hider, and then select Setup Sheet.

Every time you run the add-on, it adds 100 checkboxes. Clicking a checkbox blanks the cell to its right. Running the add-on again will generate 100 more checkboxes below.

Multiple checkboxes can be selected and then checked all at once with the space bar.

Attention

This add-on works best on blank documents.
