About uncle bobs book learnin shed

uncle bobs book learnin shed is an add-on for the Google Sheets™ web-based spreadsheet program. This tool is intended to help students learn any reading-heavy topic by using Google Sheets™.

Purpose

The intended purpose of this app is to help automate this scheduling technique by creatively combining it with this study technique, as well as by adding checkboxes to make the process use fewer clicks.
