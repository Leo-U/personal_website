Quick Start

If you'd like to quickly try out a demonstration of the app, follow this process:

1. In Google Drive, create a project folder, naming it whatever you like.

2. In that folder, create a Google Sheets file. Name it 'main', or whatever you like.

3. In that same folder, create a 'Q-A Sets' folder.

5. In that Q-A Sets folder, create three blank Google Sheets files.

6. Download each of these three CSV files.

7. In each of your three Google Sheets files from step 5, follow the below process to import each CSV file to each Google Sheets file:

a) File > Import > Upload

b) Drag each CSV file for upload, but only one at a time, not all at once.

c) Set Separator type to Custom, then copy and paste † to enter it as the Custom separator.

d) Click Import data

8. Name your Google Sheets Q-A file after whatever the CSV filename was.

9. After installing the Learning Insights add-on, open your 'main' spreadsheet from step 2. In that file, select 'Format Documents' from the 'Learning Insights' menu in the 'Extensions' menu.
