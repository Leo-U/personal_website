I built a command line chess game by following these instructions from The Odin Project, the open source coding curriculum. You can play it live by clicking this link, clicking the green 'run' button at the top of the screen, and then interacting with the terminal at the bottom of the screen. There was a problem with onlinelinegdb.com displaying the chess symbols correctly, so I had to use letters instead of piece symbols in the live version.
